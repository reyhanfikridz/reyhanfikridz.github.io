Journey of the Swordsman

Journey of the Swordsman is a third-person side-scrolling 2D action game, where players will play as a swordsman who travels around the world in search of new sword techniques. The game consists of levels where each level will have various enemies to be defeated by the player.

This was a game I created back in 5th semester of lecture as assignment to game programming courses. I was the sole creator for design and programming of this game. For other assets like music and art, I use free assets from itch.io and another website.

Technologies

Godot Game Engine Version 3.0.6
